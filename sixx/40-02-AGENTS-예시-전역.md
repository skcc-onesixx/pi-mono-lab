# 40-02. 전역 AGENTS.md 예시

**파일 위치**: `~/.pi/agent/AGENTS.md`

모든 프로젝트에서 pi를 실행할 때 공통으로 적용되는 규칙입니다.

---

```markdown
# 공통 개발 규칙

## 응답 스타일
- 응답은 한국어로 작성
- 코드 설명은 간결하게, 필요한 부분만
- 변경 전 영향 범위를 먼저 설명

## 코드 품질
- 기존 코드 스타일을 따름
- 새 파일 생성 시 프로젝트 컨벤션 확인
- 테스트 가능한 코드 작성

## 안전
- `rm -rf`, `sudo` 등 위험한 명령 전 사용자 확인
- `.env`, 비밀키 파일 수정 시 경고
```

---

**특징**:
- pi를 **어느 디렉터리**에서 실행해도 적용됨
- 프로젝트별 AGENTS.md가 있으면 **둘 다** 로드됨 (전역 → 프로젝트 순으로 이어 붙임)
